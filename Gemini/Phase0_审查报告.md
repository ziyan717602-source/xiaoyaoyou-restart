## 4. 后续建议 (Action Items for Phase 1)

Phase 0 已经打下了非常坚实的基础，可以准许进入 **Phase 1 (数据层)**。进入 Phase 1 时请注意以下几点：

1. **严格执行数据过滤**：在定义 `src/shared/types` 并加载 JSON 时，务必通过代码严格过滤出 `scope.md` 中规定的 34 个角色和 20 个怪物，切勿将全量数据加载到运行时 `G` 状态中。
2. **纯数据序列化**：在接下来的 TypeScript 类型定义中，牢记 `CLAUDE.md` 的铁律：所有装入 Boardgame.io `GameState (G)` 的结构必须是纯 JSON 可序列化的。
3. **导出脚本备份**：如果 `database-export` 脚本已被误删，建议从 Git 历史中找回并妥善存放，以防未来需要补充读取某些隐藏字段（如 `SPI` 位掩码）。
