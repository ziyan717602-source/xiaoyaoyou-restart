### 角色技能详细描述

> 触发时机编码说明（来源：Skill.cs Occurs 字段解析 + XI.cs sk02 注册表）：
> - `R#` = 自己的回合（R=Rounder，#为自己）
> - `R*` = 所有玩家的回合（*=所有人）
> - `R$` = 其他玩家的回合（$=非自己）
> - `G0` = 游戏事件（G=Game，0=本队）
> - `G1` = 游戏事件（1=全局）
> - `Z1` = 战斗开始阶段（Z=战斗，1=开始）
> - `ZD` = 战牌阶段（D=打牌）
> - `ZW` = 战斗结束阶段（W=结尾）
> - `GR` = 技牌阶段（G=技，R=回合）
> - `HD` = 宠物获得时（H=宠物，D=获得）
> - `HL` = 宠物失去时（L=离开）
> - `OH` = 受到伤害时（O=伤害，H=HP）
> - `IS` = 登场时（I=进场，S=开始）
> - `OS` = 离场时（O=出场，S=开始）
> - `ZW` = 死亡时（Z=战斗，W=死亡）
> - `TT` = 掷骰判定时（T=骰子）
> - `9P` = 命中判定时（9=命中，P=判定）
> - `CC` = 技牌使用后（C=技牌，C=结算）
> - `TH` = HP减少后（T=回合，H=HP）
> - `GE` = 技牌阶段结束（G=技，E=结束）
> - `BC` = 补牌阶段（B=补，C=开始）
> - `ST` = 回合开始阶段（S=开始，T=回合）
> - `TM` = 回合结束阶段（T=回合，M=结束）
> - `PD` = 参战时（P=参战，D=开始）
> - `ZC` = 触发战斗时（Z=战斗，C=触发）
> - `VS` = 战斗失败时（V=败，S=结束）
> - `CH` = 支援命中时（C=支援，H=命中）
> - `EV` = 事件阶段（E=事件）
> - 优先级：数值越小越先执行；负值表示在基础效果之前执行
> - `1` = 每场战斗限1次；`0` = 可多次发动

> - 第一个字母 H=HP伤害效果，T=手牌效果
